# Smart Attendance Monitoring System (SAMS)

A full-stack smart attendance system designed to automate classroom attendance using IoT devices and a cloud-based application.

## Features
• Real-time attendance tracking  
• Admin dashboard for professors  
• Flutter mobile application  
• Node.js backend API  
• MongoDB database  

## Tech Stack
Frontend: Dashboard  
Mobile: Flutter  
Backend: Node.js + Express  
Database: MongoDB
